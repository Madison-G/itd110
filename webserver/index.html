Vocabularly

Internet
A global computer network allowing a lot of information.
TCP/IP
Used to control connections of computers to the Internet. (Transmission Control Protocol/Internet Protocol)
LAN
A computer network that does not signal far. Only for local use.(Local-Area Network)
IP Address
A string of numbers that identifies computers on the Internet.
HTTP
A protocol used by the World Wide Web which defines how messages are sent. (HyperText Transfer Protocol)
Web Server
A program that used HyperText Transfer Protocol that makes web pages.
Web Client
The user on the web.
