Inline Elements

This is a test of the different types of Inline elements. Strong is when the text stands out. Bold is when the text is well, bold. Emphasis is when the words are emphasized. Italics is when the word is italisized. Quotations mark when to put quotations. Source Code indicates enclosed text that looks like source code. Keyboard indicates enclosed text that looks like a keyboard. Superscript encloses text like superscript. Subscript Encloses text like subscript. Span span.

Span does not do anything really different from what I can tell.
